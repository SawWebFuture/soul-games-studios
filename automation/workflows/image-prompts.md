## 1) LinkedIn Cover / Post Visual (Professional + Calm)

```text
Create a clean, modern editorial illustration for a LinkedIn post about [TOPIC].
Style: calm, strategic, minimal, premium startup aesthetic.
Color palette: deep navy, soft green, muted neutrals.
Composition: centered focal point, lots of negative space, subtle abstract shapes.
Mood: thoughtful, grounded, optimistic.
No text overlay, no logos, no watermark.
High detail, soft lighting, 4:5 aspect ratio.
```

## 2) X (Twitter) Visual (Bold + Fast Read)

```text
Design a striking social graphic concept for X about [TOPIC].
Style: minimal futuristic, high contrast, clean geometry.
Palette: navy, emerald, white.
Visual metaphor for: [KEY IDEA].
Keep composition simple and scroll-stopping.
No text, no watermark, no brand marks.
Sharp focus, subtle depth, 16:9 aspect ratio.
```


## 4) Instagram Reel Thumbnail (Behind-the-build)

```text
Create a cinematic yet minimal thumbnail image for a short reel about [TOPIC].
Scene should suggest building, iteration, and creative flow.
Style: modern studio desk vibe, subtle AI/futuristic cues, not sci-fi.
Color palette: navy, green accents, neutral tones.
No text, no watermark.
Vertical 9:16 composition.
```

## 5) Daily Digest Hero Image (Mind + Machine)

```text

Visual metaphor: human mind + machine intelligence in harmony.
Style: sophisticated, calm, non-hype, science-informed.

No text, no watermark.

```
