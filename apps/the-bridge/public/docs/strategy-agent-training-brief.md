# Strategy Agent Training Brief (Soul Games Studios)

## 1) Objectives
- Increase experiment shipping velocity without dropping quality.
- Improve decision quality with tight feedback loops (daily + weekly).
- Keep portfolio balanced: reliable wins + calculated bets.

## 2) Operating Framework
- **North Star:** Compounding creative output and learning speed.
- **Decision rhythm:**
  - Daily: prioritize top leverage tasks
  - Weekly: portfolio review + KPI trend review
  - Monthly: strategic reset

## 3) Core Strategic System
1. **Define one primary metric per experiment** (e.g., completion rate, saves, email opt-in).
2. **Set one hypothesis** with clear success threshold.
3. **Launch fast** with fixed scope.
4. **Review in 24–72h** and decide: scale, iterate, or kill.
5. **Log learning** in one canonical location.

## 4) Weekly Cadence
- Monday: plan portfolio (70/20/10 split)
- Tuesday–Thursday: execute + monitor
- Friday: retrospective + decision memo
- Sunday: prep next week’s top 3 priorities

## 5) KPI Design (minimum dashboard)
- Experiments launched/week
- Time-to-launch (idea → publish)
- Primary KPI hit rate
- Zone Score trend (Clarity, Speed, Energy, Quality)
- Content output consistency

## 6) Anti-Patterns
- Multiple conflicting goals per experiment
- Endless polishing before launch
- No post-launch review window
- Tracking too many vanity metrics

## 7) 30-Day Plan
- Week 1: establish baseline metrics + scorecard
- Week 2: run 3 tightly-scoped experiments
- Week 3: optimize winning format, drop weakest
- Week 4: codify SOP updates and repeat loop
