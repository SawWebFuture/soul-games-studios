# Thumbnail Templates

## General specs
- Size: 1280x720 (YouTube / Riverside export cover)
- Safe area: keep text centered with 80px margins
- Use high contrast (dark bg + bright accent)
- 2-5 words max on thumbnail text

## Text formula
- Line 1: TREND / HOOK
- Line 2: GAME MOMENT
- Optional badge: CARD ID pull / “HIDDEN GLITCH”

- Sleek sci-fi lab palette
- Add tiny glitch element in corner (brand signature)
