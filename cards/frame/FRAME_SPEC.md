# Frame Spec (v1)

## Canvas
- Final card export: **1080 x 1350** PNG
- Art panel area: **(x: 72, y: 760, w: 936, h: 430)**

## Required text areas
- Series label (top)
- Card ID / number (top-right)
- Card name (large)
- Tagline (short)
- Description (1-3 sentences)
- Footer brand/site

## Visual style
- Sleek sci-fi lab
- Dark luminous background
- Subtle metallic/glass frame
- No clutter

## Merch-safe notes
- Keep 48px safe margin from edges for print bleed tolerance
- Keep key text within center safe area
- Avoid tiny details below 2px stroke
